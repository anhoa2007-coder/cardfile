# Cardfile (Win32)
A python based Cardfile app, inspired from Windows 3.1 cardfile and note-taking app.
To run cardfile, simply download the app from the changelogs.
___
## Basic features:
It have so many basic features, such as:
- Tabbed cards
- Basic search of cards
- Duplicate cards
- Basic UI control
