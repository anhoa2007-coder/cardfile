# Cardfile (Win32)
A python based Cardfile app, inspired from Windows 3.1 cardfile and note-taking app.
To run cardfile, simply download the app from the changelogs.
___
## Basic features:
It have so many basic features, such as:
- Tabbed cards
- Basic search of cards
- Duplicate cards
- Basic UI control
___
## How do you compile it?
To compile the app:
- You need to download `pyinstaller`
- Navigate to the directory using `cd path\to\cardfile`
- And run `python -m PyInstaller --onefile --windowed --name "CardFile" main.py`
Full tutorial on `build.md`
